# Typing Speed Game

A simple typing speed game to practice typing accuracy and speed.  
Built as a front-end project to improve JavaScript, CSS and DOM manipulation skills.

## Features
- Multiple difficulty levels (Easy, Normal, Hard)
- Countdown timer per word
- Real-time score tracking
- Simple and clean UI for fast typing practice
- Restart or replay option after finishing a round

## Technologies Used
- HTML
- CSS
- JavaScript
